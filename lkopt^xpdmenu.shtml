LKOPT^XPDMENU(): Look Up Option IEN

Reference Type: Supported, Category: Menu Manager, Integration Agreement: 1157

Description

This API looks up an option's Internal Entry Number (IEN) using the "B" cross-reference.

Format

  LKOPT^XPDMENU(x)

Input Parameters

x:

(required) The name of the input option.

Output

returns:

Returns the Internal Entry Number (IEN) of the input option in the OPTION file (#19).
