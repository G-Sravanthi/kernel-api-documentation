$$VERSION^XLFIPV(): Show System Settings for IPv6

Reference Type: Supported, Category: IP Address Functions, Integration Agreement: 5844

Description

This extrinsic function determines the system settings for IPv6.

Format

$$VERSION^XLFIPV(ip)
Input Parameters:   None
Output: returns:
  • 1 — If IPv6 is enabled.
  • 0 — If IPv6 is disabled.

 

Examples

Example 1 (IPv6 Enabled)
>S X=$$VERSION^XLFIPV()
>W X
1
Example 2 (IPv6 Disabled)
>S X=$$VERSION^XLFIPV()
>W X
0
