FAIL^XTMUNIT(): Generate an Error Message

Reference Type: Not callable from outside a unit test., Category: Toolkit-M Unit Utility, Integration Agreement: N/A

Description

The FAIL^XTMUNIT API runs a test that simply generates an error message This command is useful for more complex unit tests that are built within the unit test routine itself.

Format

D FAIL^XTMUNIT(msg)

Input Parameters:

    msg:(required) The text of the error message.

Output:  returns:

      Returns

    • Returns the error message.
